Class UndoManager

Manager for handling undo and redo actions in mods. Can handle multiple undo queues.
Mods can make their own UndoActions using the provided UndoManager.IUndoAction or use the default ones Jötunn provides in UndoActions.
Undo queues get automatically reset on every login and logout.
